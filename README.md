# BLOOMQ-AI-Based-Manual-Testing-Project
## Project Overview
Bloom’s Taxonomy Analyzer is a Machine Learning and Natural Language Processing (NLP) based application that classifies educational text according to Bloom’s Taxonomy cognitive levels.

The project was developed using Python and the Scikit-learn library to analyze and categorize text into different Bloom’s levels such as Remembering, Understanding, Applying, Analyzing, Evaluating, and Creating.

This repository contains the complete Manual Testing documentation for the project.
